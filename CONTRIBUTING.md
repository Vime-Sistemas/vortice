Contribuindo para o Vortice

Obrigado pelo interesse em contribuir! Algumas orientações para acelerar a aceitação da sua PR:

- Faça um fork do repositório e crie uma branch para sua alteração.
- Sempre escreva testes para novas funcionalidades e correções de bugs.
- Mantenha as mudanças focadas e documente o comportamento no `README.md`.
- Execute `go test ./...` e verifique se todos os testes passam antes de submeter.
- Abra um Pull Request descrevendo a mudança, o motivo e qualquer impacto esperado.

Para mudanças maiores, abra uma issue antes para discutir o design.

Obrigado! Seu tempo e contribuições tornam o projeto melhor para todos.
